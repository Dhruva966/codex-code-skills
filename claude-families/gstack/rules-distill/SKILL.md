# Rules Distill

When several skills keep teaching the same principle, elevate it.

## Workflow

1. inventory skills and current rules
2. identify principles that appear in multiple places
3. check whether the principle is already covered
4. propose one of:
   - append to existing rule
   - revise an outdated rule
   - add a new section
   - keep it skill-specific

## Good Candidates

- repeated execution discipline
- repeated review or verification habits
- repeated safety or scope rules

## Rule

Do not invent new shared rules from one-off opinions. Distill only patterns with real cross-skill evidence.
