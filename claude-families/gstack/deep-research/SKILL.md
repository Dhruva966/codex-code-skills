# Deep Research Report

Produce a multi-source research memo, not a vibes summary.

## When To Use

- competitive analysis
- current-state landscape review
- due diligence on companies, products, or technologies
- evidence-backed recommendation with tradeoffs
- broad “research this deeply” asks

## Workflow

1. Break the topic into sub-questions.
   - what is it
   - who matters
   - what changed recently
   - risks, tradeoffs, or gaps

2. Search broadly, then read narrowly.
   - gather multiple sources
   - favor official, primary, recent, and reputable sources
   - read a smaller set deeply before writing

3. Cross-check important claims.
   - if only one source says it, label it as thinly supported
   - include dates for freshness-sensitive facts

4. Write a report with evidence boundaries.
   - executive summary
   - major themes
   - key takeaways
   - sources
   - methodology

## Report Standard

Every important claim should be one of:

- sourced fact
- user-provided context
- inference
- recommendation

Do not blur them together.
