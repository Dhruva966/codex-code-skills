# Session Handoff

Leave the next session a clean trail, not a mystery.

## Include

- goal and current status
- decisions made
- files changed or most relevant
- verification already run
- blockers or unresolved questions
- next smallest useful step

## Rules

- keep it concise
- prefer concrete state over narrative
- distinguish finished work from intended work
- call out anything time-sensitive or risky

This is especially useful when stopping midstream or after a broad investigation.
