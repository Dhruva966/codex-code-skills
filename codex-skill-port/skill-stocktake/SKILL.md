# Skill Stocktake

Audit the skill collection like a maintainer, not a collector.

## What To Check

- overlap with other skills
- stale or time-sensitive references
- trigger quality and scope fit
- whether the skill changes behavior or just repeats defaults
- whether a skill should be kept, improved, merged, or retired

## Verdicts

- Keep
- Improve
- Update
- Merge
- Retire

## Rules

- evaluate the actual value of the skill, not just its existence
- prefer fewer sharper skills over many overlapping ones
- be explicit about what would replace a retired or merged skill
